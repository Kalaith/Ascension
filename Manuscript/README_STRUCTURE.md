# Manuscript Folder Structure

## Novel
- `Novel/Drafts/`: Core story drafts (prequel + parts/chapters).

## Supporting
- `Supporting/Worldbuilding/`: Lore, world notes, character notes, and planning docs.
- `Supporting/Feedback/`: Critiques, beta-reader notes, and revision comments.
- `Supporting/Reference/`: Research snippets, links, and background material.

## Exports
- `Exports/PlainText/`: Generated plain-text exports from `.rtf` source files.

## Tools
- Run `Tools/strip-rtf-formatting.ps1` to convert `.rtf` to plain text.
- Default output target is `Manuscript/Exports/PlainText`.
